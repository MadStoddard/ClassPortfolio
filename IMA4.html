IMA4- Interactive ELearning Lesson Development

IMA4- Protecting Ideas: An Interactive Intro to Intellectual Property

This self-paced online training module introduces learners to the basics of Intellectual Property (IP), including patents, trademarks, copyrights, and trade secrets. Built using Adobe Captivate, the course includes instructional slides, a formative quiz, and an interactive scenario designed to reinforce key concepts through real-world application. This project showcases my skills in e-learning design, interactive media, and instructional sequencing for digital environments.
